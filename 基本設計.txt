【障害者支援利用計画自動作成システム　基本設計書】

■ システム概要
本システムは、障害者の支援利用計画作成業務を効率化するため、音声データから重要情報を自動抽出・分類し、雛形に自動出力するWebアプリケーションです。

■ 機能一覧
1. ダッシュボード
   - 計画書作成状況の統計表示
   - 最近の計画書一覧
   - クイックアクション
2. 音声アップロード
   - 音声ファイルのアップロード（MP3, WAV, M4A, OGG対応）
   - アップロード進捗表示
3. 計画書一覧
   - 作成済み計画書の一覧表示
   - 検索・フィルタリング
   - ページネーション
4. 計画書編集
   - ステップ形式での編集（基本情報、現状・アセスメント、支援目標、サービス利用計画、支援体制、モニタリング計画）
   - リアルタイム保存
5. 計画書詳細表示
   - 完成計画書の詳細表示
   - PDF出力・印刷

■ 画面構成
- ダッシュボード画面
- 音声アップロード画面
- 計画書一覧画面
- 計画書編集画面
- 計画書詳細画面

■ データ構造（例）
- User（ユーザー情報）
  - id, name, email, role
- Plan（計画書）
  - id, user_id, status, created_at, updated_at
- AudioFile（音声ファイル）
  - id, plan_id, file_path, uploaded_at
- Assessment（アセスメント情報）
  - id, plan_id, content, created_at

■ 使用技術
- フロントエンド: React.js + TypeScript
- UI: Material-UI (MUI)
- ルーティング: React Router
- 状態管理: React Query
- バックエンド: Supabase（予定）
- 音声認識: Google Cloud Speech-to-Text API（予定）
- AI: Google Gemini API（予定）
